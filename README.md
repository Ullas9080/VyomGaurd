# VyomGarud - Next-Gen UAV Systems Landing Page

VyomGarud is a modern, dark-themed landing page for a company specializing in aerial defense and surveillance UAV systems. The page showcases the company's mission, capabilities, and key highlights in a futuristic, high-tech design.

## Features

- **Hero Section:** Full-screen background video with animated headings and slogans.  
- **About Section:** Informative cards about VyomGarud’s mission, AI-driven UAV platforms, and autonomous systems.  
- **Capabilities / Products:** Cards showcasing products and technical strengths with icons and hover effects.  
- **Highlights:** Concise 3-point features section with gradient cards and futuristic styling.  
- **Footer:** Dark-themed footer with branding, quick links, social icons, and contact form.  
- **Responsive Design:** Fully responsive for mobile, tablet, and desktop screens.  
- **Futuristic UI:** Dark, military-inspired theme with orange accents, gradients, and hover animations.
